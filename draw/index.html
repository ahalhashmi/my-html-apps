Shapes

Rectangle, ellipse, and line tools draw onto the active layer. With auto-layer enabled, each new shape gets its own transparent layer so you can move, resize, and rotate it later.

Offset Point

Set a visible point on the canvas. Then draw somewhere else: the stroke appears on that point and follows your finger movement. The point follows the stroke and stays at the last drawn end point.
No offset point

Touch

When enabled, multiple fingers can draw at the same time. Offset point mode and multi-touch cannot be enabled together. Transform mode still supports two-finger pinch/rotate for the active layer.
